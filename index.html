iOS App

Working on a user friendly app under the IOS framework aimed at improving typical U.S. Federal and Local voting procedures by incorporating a program that executes a ranked-choice voting system. The API app uses its backend logic to determine the winner when there is no clear majority. The front end user interface aims to create an intuitive platform to represent voter preference more holistically than a typical plurality election.
